P0022

Code defaut P0022 — Calage arbre a cames admission retard excessif banc 2

Que signifie le code P0022 ?

Le code P0022 indique que l arbre a cames d admission du banc 2 ne recule pas assez rapidement. Equivalent de P0012 mais pour le banc 2 sur moteurs V6/V8. Commun sur BMW, Audi, Toyota.

Causes les plus frequentes

  • Solenoide VVT banc 2 encrase ou defaillant
  • Huile trop visqueuse ou tres encrassee
  • Filtre a huile VVT bouche
  • Phaseur arbre a cames banc 2 use
  • Probleme circuit hydraulique distribution

Que faire ?

Comme P0012, un changement d huile avec vidange de l ancien solvant resout souvent ce code sur les vehicules recents. MecaIA verifie l historique entretien et guide vers la cause reelle.
