P0087

Code P0087 — Pression carburant trop basse rampe

Description

Le P0087 indique une pression trop basse dans la rampe d injection. Sur diesel common-rail, ce code peut arreter le moteur en protection. Pression requise : 200 a 2000 bar selon le regime.

Symptomes observables

  • Demarrage difficile ou impossible
  • Coupures moteur en charge
  • Mode degrade automatique
  • Fumees noires diesel
  • Consommation irreguliere

Causes les plus frequentes

  • Pompe haute pression usee
  • Regulateur de pression defaillant
  • Filtre carburant colmate
  • Fuite haute pression injecteur rail
  • Soupape regulation rail grippee
