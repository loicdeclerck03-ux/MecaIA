P0171

Code défaut P0171 — Mélange trop pauvre banc 1

Qu'est-ce que le code P0171 ?

Le code P0171 signale que le mélange air/carburant du banc 1 est trop pauvre (trop d'air, pas assez de carburant). L'IA détecte cette dérive via le fuel trim LTFT. Causes fréquentes : sonde MAF sale, fuite d'air, pompe à carburant faible.

Causes les plus fréquentes

  • Sonde MAF encrassée
  • Fuite d'air admission
  • Pompe carburant affaiblie
  • Injecteurs bouchés
  • Filtre carburant colmaté

Que faire ?

Ne roulez pas longtemps avec ce code actif. Commencez par scanner l'ensemble des systèmes de votre voiture pour avoir une image complète. MecaIA analyse vos données OBD en temps réel et vous guide étape par étape vers la cause exacte du problème.
