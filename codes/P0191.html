P0191

Code defaut P0191 — Performance capteur pression rampe carburant

Qu’est-ce que le code P0191 ?

Le code P0191 indique que la pression de rampe carburant varie en dehors des limites attendues (performance). Different de P0087/P0193 qui sont des valeurs fixes — P0191 indique une pression instable.

Causes les plus frequentes

  • Regulateur pression carburant defaillant
  • Pompe carburant fatiguee (debit insuffisant)
  • Injecteurs qui fuient
  • Capteur pression rampe defectueux
  • Rail d'injection partiellement obstrue

Que faire ?

P0191 est souvent un signe de fatigue pompe carburant. MecaIA analyse la pression en live sous differentes charges pour evaluer si c'est la pompe, les injecteurs ou le capteur.
