Code défaut P0300 — Ratés d'allumage aléatoires

Qu'est-ce que le code P0300 ?

Le code P0300 indique des ratés d'allumage aléatoires sur plusieurs cylindres. Le moteur tremble, la consommation augmente et le voyant moteur clignote. Cause fréquente : bougies usées, bobines d'allumage défectueuses, injecteurs encrassés.

Causes les plus fréquentes

  • Bougies d'allumage usées
  • Bobines d'allumage défectueuses
  • Injecteurs encrassés
  • Fuite de collecteur admission
  • Compression faible

Que faire ?
