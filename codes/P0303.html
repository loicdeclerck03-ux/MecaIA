P0303

Code P0303 — Rates d allumage cylindre 3

Description

Le P0303 signale des rates d allumage anormaux sur le cylindre 3. Non traite, ce code deteriore le catalyseur et peut endommager le moteur.

Symptomes observables

  • Vibrations perceptibles au ralenti
  • Manque de puissance
  • Consommation excessive
  • Odeur carburant cru a l echappement
  • Voyant moteur clignotant

Causes les plus frequentes

  • Bougie defaillante ou usee sur cyl. 3
  • Bobine d allumage defectueuse
  • Injecteur encrase ou defaillant
  • Basse compression soupapes soupapes
  • Fuite joint de culasse
