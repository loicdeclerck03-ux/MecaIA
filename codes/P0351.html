P0351

Code defaut P0351 — Circuit bobine allumage cylindre 1

Que signifie le code P0351 ?

Le code P0351 indique un probleme sur le circuit de commande de la bobine d allumage du cylindre 1. Symtomes similaires a P0301 : rates, vibrations, voyant clignotant.

Causes les plus frequentes

  • Bobine allumage cylindre 1 defaillante
  • Cablage ou connecteur bobine endommage
  • Driver bobine dans l ECU defaillant
  • Court-circuit ou circuit ouvert
  • Bougie cylindre 1 en court-circuit

Que faire ?

P0351 est un code circuit (electrique), different de P0301 qui est le symptome (rate). Les deux ensemble = confirme probleme bobine/cylindre 1. MecaIA distingue les deux et guide vers le composant.
