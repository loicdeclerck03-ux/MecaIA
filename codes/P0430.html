P0430

Code defaut P0430 — Catalyseur banc 2 rendement insuffisant

Que signifie le code P0430 ?

Le code P0430 indique que le catalyseur du banc 2 (cote droit sur un V6/V8) est en dessous du seuil de rendement minimum. Symptomes : voyant moteur allume, odeur soufre possible, echec CT emissions.

Causes les plus frequentes

  • Catalyseur banc 2 use ou empoisonne
  • Sonde lambda aval banc 2 defaillante
  • Fuite echappement avant la sonde aval
  • Huile dans le circuit de combustion (joints de culasse)
  • Carburant non brule (rates d allumage non traites)

Que faire ?

Un P0430 seul sans autres codes = probablement catalyseur a remplacer. Avec P0303/P0304 = traiter les rates d abord. MecaIA croise avec les donnees live pour etablir le diagnostic.
