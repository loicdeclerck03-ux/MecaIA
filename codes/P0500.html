P0500

Code P0500 — Capteur vitesse vehicule VSS

Description

Le P0500 signale que l ECU ne recoit plus de signal coherent du capteur de vitesse vehicule. Ce signal est essentiel pour le regulateur de vitesse et la boite automatique.

Symptomes observables

  • Compteur vitesse inactif ou erratique
  • Regulateur vitesse desactive
  • Boite auto passages anormaux
  • Consommation legerement degradee

Causes les plus frequentes

  • Capteur VSS defaillant sur boite
  • Roue phonique ABS endommagee
  • Cablage connecteur VSS coupe oxyde
  • Module ABS defaillant
  • Pneumatique de taille incorrecte
