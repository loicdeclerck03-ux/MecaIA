P0520

Code defaut P0520 — Circuit capteur pression huile moteur

Que signifie le code P0520 ?

Le code P0520 indique un probleme sur le circuit du capteur de pression d huile moteur. ATTENTION : ce code ne signifie pas forcement que la pression d huile est faible — mais il faut verifier immediatement pour eviter tout risque moteur.

Causes les plus frequentes

  • Capteur pression huile defectueux
  • Cablage ou connecteur capteur endommage
  • Pression d huile reellement faible (urgence)
  • Joint capteur huile fuite
  • Calculateur moteur defaillant

Que faire ?

SI le voyant huile clignote en meme temps, ARRETEZ le moteur immediatement. Si c est uniquement un code P0520 sans symptome, c est probablement le capteur lui-meme. MecaIA evalue la gravite selon les conditions.
