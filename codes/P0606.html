P0606

Code P0606 — Defaillance interne ECU PCM

Description

Le P0606 signale que l ECU a detecte une erreur dans son propre processeur. Souvent declenche a tort par des problemes de tension. Verifiez alimentation et masses avant tout.

Symptomes observables

  • Moteur mode degrade
  • Multiples codes erreur inexpliques
  • Perte communication outil diagnostic
  • Coupures moteur soudaines

Causes les plus frequentes

  • Tension alimentation insuffisante
  • Masse ECU corrodee
  • Surtension alternateur defaillant
  • Eau dans le calculateur
  • Defaillance hardware interne rare
