Code defaut P0715 — Circuit vitesse arbre primaire turbine

Que signifie le code P0715 ?

Le code P0715 concerne uniquement les vehicules a boite automatique. Il indique un probleme sur le capteur de vitesse de l arbre d entree de la turbine. Symptomes : a-coups, changement de vitesse au mauvais regime, boite en mode securite.

Causes les plus frequentes

  • Capteur vitesse turbine (TSS) defaillant
  • Cablage capteur endommage
  • Debris metalliques sur le capteur magnetique
  • Niveau huile boite insuffisant
  • Probleme interne boite automatique

Que faire ?

P0715 peut declencher le mode limp home de la boite automatique. Verifier d abord le niveau et la qualite de l huile boite. MecaIA evalue la gravite et si une intervention boite est necessaire.
