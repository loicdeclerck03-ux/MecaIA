C'est quoi le diagnostic OBD2 ?

L'OBD2 (On-Board Diagnostics 2) est le systeme informatique embarque obligatoire dans toutes les voitures vendues en Europe depuis 2001 (essence) et 2004 (diesel). Il surveille en permanence le moteur, la transmission, les emissions et de nombreux autres systemes.

Quand un capteur detecte une anomalie, le calculateur stocke un code defaut DTC (Diagnostic Trouble Code) et allume le voyant moteur. Ces codes suivent une norme internationale : P0300 signifie toujours « rates d'allumage », peu importe la marque ou le pays.

💡 Bon a savoir : Le voyant moteur peut s'allumer pour une sonde lambda a 40€ comme pour un probleme de turbo a 1200€. Seul le code DTC exact permet de savoir ce qui se passe vraiment.

Les 4 categories de codes DTC

🟡

P — Moteur/Transmission

Codes les plus frequents. Moteur, boite, emissions. Ex : P0171 (melange pauvre).

🔵

B — Carrosserie

Airbags, climatisation, sieges electriques. Ex : B0100.

🟠

C — Chassis

Freins ABS, direction assistee, suspension active. Ex : C0040.

⚪

U — Reseau CAN

Communication entre calculateurs. Ex : U0100 (perte com moteur).

Mettez le contact (sans demarrer)

Le calculateur s'active au contact. Pas besoin de demarrer le moteur pour lire les codes defauts.
