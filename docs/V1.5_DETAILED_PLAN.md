## 🚀 V1.5 (2-3 semaines après lancement V1.0)

### Priorité 1️⃣ : ANALYSE DE DEVIS (KILLER FEATURE)
**Valeur** : Viral, différencie MecaIA de tout chatbot, défend le client
**Risque** : Base de prix doit être fiable ou l'IA hallucine
**Implémentation** :

```
1. Importér seed data (repair_prices_reference.json + invoice_examples.json)
2. Ajouter route `/api?action=analyze_invoice`
   - User upload PDF/texte devis
   - Claude Sonnet compare aux patterns + fourchettes
   - Retour : "Verdict: SUSPECT | Surcharge détectée: +45%"
3. UI : afficher breakdown (pièces vs main d'œuvre vs surcoûts)
4. RED FLAGS : lister automatiquement (biellettes +150%, labor 4h)
5. Tests : 10 cas réels (suspect/raisonnable)
```

**Temps estimé** : 8 heures (code + tests)
**Priorité absolue** : OUI (c'est ton unique selling point viral)

### Priorité 2️⃣ : DIAGNOSTIC GUIDÉ + SUIVI GRATUIT
**Valeur** : Précision accrue, meilleure UX, rétention (crédits moins effrayants)
**Implémentation** :

```
1. Diagnostic guidé : l'IA pose questions avant de conclure
   - "À froid ou à chaud ?" → "À chaque démarrage ?" → narrowing
   - Claude Haiku = économique pour les questions
   - Fin : "Je suis à 85% certain c'est X"

2. Suivi gratuit 15 min après diagnostic payant
   - Crédit payé pour le diagnostic
   - 15 min = suivi gratuit limité au contexte
   - Règle : suivi ≠ nouveau problème (sinon = nouveau crédit)
   - Fenêtre : 24h post-diagnostic pour poser questions
   - UI : timer visible ("6 min restantes sur ce suivi gratuit")

3. Tests : vérifier que suivi n'expose pas le "gratuit illimité"
```

**Temps estimé** : 6 heures (prompt + timeout + UI)
**Priorité absolue** : OUI (valorise l'offre, réduit stress crédit)

### Priorité 3️⃣ : PLAN ENTRETIEN PRÉDICTIF
**Valeur** : Moins urgent que devis/suivi, mais logique
**Implémentation** :

```
1. Utiliser vehicle_models_common_failures.json
2. Route : `/api?action=predictive_maintenance`
   - Input : modèle voiture + kilométrage + âge
   - Output : "À 100k km, attention suspension. À 120k, timing critique"
3. Alertes : "Ta Clio approche 120k → timing dans 3 mois"
4. Intégrer dans dashboard utilisateur
```

**Temps estimé** : 4 heures
**Priorité** : Après devis + suivi gratuit

## 🔒 V1.5 — GARDE-FOUS CRITIQUES

### Mode "Comment réparer ?" — À **REPOUSSER** en V2
**Raison** : Risque sécurité/légal très réel (hallucinations géométriques, airbag, freins)
**Implémentation correcte** (V2, encadré) :
```
- NE PAS générer les étapes (trop risqué)
- POINTER vers sources fiables : RTA + vidéos YouTube officielles
- INTERDIRE : freins, direction, airbag, distribution, haute tension
- Décliner de responsabilité : "Ce n'est pas un guide DIY complet"
```

### Sécurité analyse devis
```
- JAMAIS faire confiance totalement au prix généré par IA
- Afficher disclaimer : "Fourchettes indicatives. Consulte un garage local"
- RED FLAGS en gras (surcoût > 50%)
```
