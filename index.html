Questions fréquentes
Comment fonctionnent les crédits ?▼
1 crédit = 1 diagnostic OBD, 1 recherche pièces, 1 alerte entretien ou 1 analyse photo. Le décodeur VIN est toujours gratuit. Les crédits n'expirent jamais sauf le pack illimité (30 jours).
Les diagnostics sont-ils fiables ?▼
L'IA analyse le code OBD, les symptômes et le véhicule pour un diagnostic précis. Plus vous donnez d'infos, plus c'est précis. Les résultats sont indicatifs — consultez un professionnel pour les interventions critiques.
Comment fonctionne le garage virtuel ?▼
Ajoutez vos véhicules une seule fois. L'IA les mémorise et les utilise automatiquement pour tous vos diagnostics. Sauvegardé dans votre compte sécurisé — accessible depuis n'importe quel appareil.
Le paiement est-il sécurisé ?▼
100% sécurisé par Stripe, leader mondial du paiement en ligne. Votre carte n'est jamais stockée sur nos serveurs. Crédits activés immédiatement après confirmation bancaire.
Les crédits expirent-ils ?▼
Non ! Ils n'expirent jamais sauf le pack illimité (30 jours). Achetez quand vous en avez besoin.
Comment fonctionnent les codes promo ?▼
Deux types : réduction % (ex: -20% sur un pack) et crédits gratuits. Entrez le code dans le modal de paiement. Générés par Loïc Declerck et partagés via partenaires et influenceurs.
MecaIA fonctionne sur mobile ?▼
Oui — optimisé pour iPhone et Android. Installez-le comme une vraie app via "Ajouter à l'écran d'accueil".
