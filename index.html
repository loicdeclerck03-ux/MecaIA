Questions fréquentes
Comment fonctionnent les crédits ?▼
1 crédit = 1 diagnostic OBD, 1 recherche pièces, 1 alerte entretien ou 1 analyse photo. Le décodeur VIN est toujours gratuit. Les crédits n'expirent jamais sauf le pack illimité (30 jours).
Les diagnostics sont-ils fiables ?▼
L'IA analyse le code OBD, les symptômes et le véhicule pour un diagnostic précis. Plus vous donnez d'infos, plus c'est précis. Les résultats sont indicatifs — consultez un professionnel pour les interventions critiques.
Comment fonctionne le garage virtuel ?▼
Ajoutez vos véhicules une seule fois. L'IA les mémorise et les utilise automatiquement pour tous vos diagnostics. Sauvegardé dans votre compte — accessible depuis n'importe quel appareil.
Comment fonctionnent les codes promo ?▼
Deux types : réduction % (ex: -20% sur un pack) et crédits gratuits. Entrez le code dans le modal de paiement. Partagés via partenaires et influenceurs.
Mes données sont-elles sécurisées ?▼
Vos données sont chiffrées et stockées sur Supabase (infrastructure PostgreSQL sécurisée). Aucune donnée n'est partagée avec des tiers. Vous pouvez supprimer votre compte à tout moment.
Combien de crédits faut-il pour un diagnostic ?▼
1 crédit = 1 session de diagnostic complète avec Dylan. Une session dure en moyenne 3-5 échanges jusqu'à la conclusion. Le décodeur VIN est gratuit (3 par jour). La session est valable 7 jours : vous pouvez la reprendre à tout moment.
Dylan peut-il se tromper dans son diagnostic ?▼
Dylan est un outil d'aide au diagnostic, pas un remplacement au mécanicien. Il identifie les causes probables avec un niveau de confiance clairement indiqué (faible/probable/forte/très forte). Pour les interventions critiques (freins, direction, haute tension), consultez toujours un professionnel qualifié.
Les références de pièces sont-elles garanties exactes ?▼
Les références sont générées par IA à titre indicatif. Vérifiez toujours la compatibilité exacte sur Autodoc, Oscaro ou chez votre fournisseur avant commande. Les prix sont des fourchettes indicatives — les prix réels varient selon les fournisseurs.
Comment obtenir le carnet d'entretien de mon véhicule ?▼
Sélectionnez votre véhicule dans le garage, puis tapez simplement à Dylan : « carnet d'entretien » ou « quand changer mon huile ? ». Dylan affiche instantanément les intervalles officiels (vidange, distribution, liquides, pression pneus) sans consommer de crédit.
