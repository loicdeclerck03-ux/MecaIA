Questions fréquentes
Comment fonctionnent les crédits ?▼
1 crédit = 1 diagnostic OBD, 1 recherche pièces, 1 alerte entretien ou 1 analyse photo. Le décodeur VIN est toujours gratuit. Les crédits n'expirent jamais sauf le pack illimité (30 jours).
Les diagnostics sont-ils fiables ?▼
L'IA analyse le code OBD, les symptômes et le véhicule pour un diagnostic précis. Plus vous donnez d'infos, plus c'est précis. Les résultats sont indicatifs — consultez un professionnel pour les interventions critiques.
Comment fonctionne le garage virtuel ?▼
Ajoutez vos véhicules une seule fois. L'IA les mémorise et les utilise automatiquement pour tous vos diagnostics. Sauvegardé dans votre compte — accessible depuis n'importe quel appareil.
Comment fonctionnent les codes promo ?▼
Deux types : réduction % (ex: -20% sur un pack) et crédits gratuits. Entrez le code dans le modal de paiement. Partagés via partenaires et influenceurs.
Mes données sont-elles sécurisées ?▼
Vos données sont chiffrées et stockées sur Supabase (infrastructure PostgreSQL sécurisée). Aucune donnée n'est partagée avec des tiers. Vous pouvez supprimer votre compte à tout moment.
Conditions d'utilisation

Service — MecaIA est un outil d'aide au diagnostic automobile basé sur l'IA. Les résultats sont indicatifs et ne remplacent pas l'avis d'un professionnel qualifié.

Responsabilité — MecaIA ne peut être tenu responsable des décisions prises sur la base des diagnostics. Consultez toujours un mécanicien professionnel pour les interventions critiques.
